# Mental Health Counseling Portal

A comprehensive, calming mental health counseling website built with HTML, CSS, and JavaScript. This website provides information about mental health services, therapist profiles, resources, and appointment booking.

## Features

- **7 Complete Pages:**
  - Homepage (index.html)
  - About Page (about.html)
  - Services Page (services.html)
  - Therapists Page (therapists.html)
  - Therapist Profile Page (therapist-profile.html)
  - Resources Page (resources.html)
  - Contact Page (contact.html)

- **Design System:**
  - Calming color palette (Deep Teal Blue, Soft Aqua, Mint Cream)
  - Professional typography (Merriweather, Open Sans, Lora)
  - Watercolor-inspired backgrounds
  - Smooth animations and transitions
  - Responsive design for all devices

- **Interactive Features:**
  - Testimonial carousel
  - Filter functionality for therapists
  - Accordion FAQ sections
  - Smooth scroll navigation
  - Form validation
  - Mobile-responsive navigation

## Customization

To customize the site:

1. **Colors:** Update CSS variables in `css/styles.css`
2. **Content:** Edit HTML files directly
3. **Styling:** Modify `css/styles.css`
4. **Functionality:** Update `js/main.js`
